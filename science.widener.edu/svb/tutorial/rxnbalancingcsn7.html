Balancing Chemical Reactions

This page provides exercises in balancing chemical reactions. When you press "New Reaction", a chemical equation with an empty cell before each compound will be displayed below. Enter the appropriate coeffcients in each cell and press "Check Answer." Results will appear immediately in the scoring table. This page will only accept the smallest set of integer coefficients.
  • If you miss a problem three times, pressing "Show Answer" will display the complete solution and you will no longer be able submit an answer for that problem.
