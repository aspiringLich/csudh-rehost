Formulas and Per Cent Composition

This page permits you to practice/check your Mass % and formula calculations. At this time, you are limited to using only the elements C,H,N and O. That will change shortly, but should be sufficient for you to build your skills. The page can take the following inputs:
  • Input any set of mass %s, which add to 100, but do not provide a gfw. The program will return the empirical formula.
  • Input the mass %s plus the gfw and the page will return the molecular formula.
  • Input the mole quantities and the program will return the mass %s and the gfw.

Additional Points:

  • It is strongly recommended that you clear extra data before running a problem. To help in this:
    • Pressing "Clear 1" will clear the first row only.
    • Pressing "Clear 2" will clear the second row only.
  • Pressing "Clear All" clears the entire form.
  • Pressing "Calculate" processes the input.
